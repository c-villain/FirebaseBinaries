## Swift Package Manager

To integrate `FirebaseBinaries` into your project using SwiftPM add the following to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/c-villain/FirebaseBinaries", from: "9.2.0"),
],
```
